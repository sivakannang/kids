# Colors Words - Theme Based Reading

**Focus**: Color names and color-related vocabulary
**Goal**: Recognize colors instantly

---

## Basic Colors

```
red    blue   green  yellow  orange
black  white  brown  pink    purple
gray   gold   silver
```

### Read Aloud

- **red** - bright color
- **blue** - cool color
- **green** - color of leaves
- **yellow** - bright like sun
- **orange** - color of orange fruit
- **black** - darkest color
- **white** - lightest color
- **brown** - dark earthy color
- **pink** - light red
- **purple** - mix of red and blue
- **gray** - mix of black and white
- **gold** - bright like treasure
- **silver** - shiny like metal

### Small Sentences

```
The apple is red.
The sky is blue.
Grass is green.
The sun is yellow.
The orange fruit is orange.
Coal is black.
Snow is white.
The tree is brown.
The flower is pink.
Grapes are purple.
The road is gray.
Gold is shiny.
Silver is bright.
```

## Things & Their Colors

```
apple is red
sky is blue
grass is green
sun is yellow
flower is red/pink
```

### Read Aloud

Practice associating objects with their colors:
- Apple is **red**
- Sky is **blue**
- Grass is **green**
- Sun is **yellow**
- Flower can be **red**, **pink**, **yellow**

### Small Sentences

```
The red apple is sweet.
The blue sky is beautiful.
The green grass is soft.
The yellow sun is hot.
The pink flower is pretty.
White clouds are fluffy.
Brown tree is big.
Black coal is dark.
```

## Color Descriptions & Mixing

```
light   dark    bright  pale
light blue    light green    light pink
dark red      dark blue      dark green
bright red    bright yellow
```

### Read Aloud

- **light** - pale/not dark
- **dark** - deep color
- **bright** - vivid/shiny
- **pale** - very light

### Small Sentences

```
Light blue is pale.
Dark blue is deep.
Bright red is vivid.
Light green is soft.
Dark green is deep.
Bright yellow is hot.
Pale pink is soft.
```

## Our World in Colors

```
rainbow  color  paint  draw  colorful
```

### Read Aloud

- **rainbow** - all colors in sky after rain
- **color** - different hues
- **paint** - liquid color
- **draw** - make with pen/pencil
- **colorful** - many colors

### Small Sentences

```
The rainbow has many colors.
I like all colors.
Paint with bright colors.
Draw a colorful picture.
The garden is colorful.
```

## Reading Practice - Colors Theme

### Read Aloud (Slow)

**Part 1: Color Names**

```
Red is bright.
Blue is cool.
Green is from leaves.
Yellow is from sun.
Orange is from fruit.
```

**Part 2: Things and Colors**

```
The apple is red.
The sky is blue.
Grass is green.
The sun is yellow.
Snow is white.
```

**Part 3: More Colors**

```
Black is dark.
Brown is earthy.
Pink is light red.
Purple is mix of red and blue.
Gray is light black.
```
