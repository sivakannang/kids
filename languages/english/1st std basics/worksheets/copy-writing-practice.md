## How to Use This Page

1. **Point and read** the word/sentence first
2. **Say it aloud**
3. **Copy it** on the line provided
4. **Check your work** - does it match?
5. **Practice again** if needed

## Handwriting Tips

1. **Sit properly** - Back straight, feet on floor
2. **Hold pen correctly** - Not too tight, not too loose
3. **Space letters** - Not too close, not too far
4. **Make letters clearly** - Easy to read
5. **Write on line** - Use the lines as guide
6. **Copy letter by letter** - Don't guess
7. **Check after** - Does it match the original?

## Mistakes & Learning

**It's okay to make mistakes!**

When you copy:
- If letter is wrong, cross it and write again
- If whole word is wrong, try again
- If too much wrong, start fresh on next line

Example:
```
Original:  I can run
Your try:  I can rwm  (mistake)
Correct:   I can run
```
